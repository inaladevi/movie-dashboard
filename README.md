# 🎬 NETFLEX Dashboard

A movie streaming dashboard inspired by modern platforms like Netflix.  
This project focuses on layout architecture using CSS Grid, reusable components, horizontal content sliders, and interactive UI elements.

The dashboard displays curated movie sections with smooth horizontal scrolling and a side panel showing upcoming releases.

## Live Demo

[View on GitHub Pages](https://inaladevi.github.io/admin-dashboard/)

## Features

- Dashboard layout built with **CSS Grid**
- Fixed sidebar navigation with icons
- Top header including:
  - Greeting section
  - Search bar with icon
  - Notification icon
  - User profile section
- Multiple movie sections:
  - **Recently Watched**
  - **Trending Now**
  - **Recommended**
- Horizontal scrolling movie rows with arrow controls
- Interactive movie cards with:
  - Rating badges
  - Hover effects
  - Play and information icons on hover
- Right-side discovery panel including:
  - **New Releases**
  - **Coming Soon**
- Scrollable movie content while keeping sidebar and header fixed
- Clean dark theme inspired by streaming platforms

## Built With

- **HTML5** (Semantic layout)
- **CSS3**
  - CSS Grid
  - Flexbox
  - Custom Properties (CSS Variables)
  - Hover interactions
- **JavaScript**
  - Horizontal slider functionality using `scrollBy()`
- **Google Fonts**
